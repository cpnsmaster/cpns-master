# 1. Design System Overview

UI/UX Design System CPNS Master adalah pedoman visual dan interaksi yang digunakan untuk menjaga konsistensi seluruh aplikasi.

Design System ini menjadi acuan untuk:

- Website publik.
- Dashboard pengguna.
- Halaman belajar.
- Halaman latihan soal.
- Simulasi CAT.
- Admin CMS.

Tujuan utama:

> Membuat CPNS Master terlihat profesional, modern, mudah digunakan, dan konsisten.

# 10. Spacing System

```text

16px  = lg
24px  = xl

48px  = 3xl

```

## Recommended Usage

```text
Icon gap       → 8px
Input padding  → 12px - 16px
Card padding   → 20px - 24px
Section gap    → 32px - 64px
Page padding   → 24px - 48px
```

# 30. UX Principles

CPNS Master mengikuti prinsip:

## 30.1 Clarity

Pengguna harus selalu memahami:

- Sedang berada di halaman apa.
- Apa yang harus dilakukan.
- Apa yang terjadi setelah action.

## 30.2 Consistency

Komponen yang sama harus memiliki perilaku yang sama.

## 30.3 Feedback

Setiap action penting harus memberikan feedback.

## 30.4 Simplicity

Jangan menambahkan kompleksitas yang tidak diperlukan.

## 30.5 Progress

Pengguna harus dapat melihat perkembangan mereka.

# 31. Design Tokens

Design tokens akan digunakan sebagai sumber utama untuk styling.

Contoh:

```typescript
const designTokens = {
  colors: {
    primary: "#F5B800",
    background: "#0A0A0A",
    surface: "#171717",
    text: "#FFFFFF",
  },
  radius: {
    sm: "6px",
    md: "10px",
    lg: "16px",
  },
};
```

Pada implementasi sebenarnya, token akan disesuaikan dengan konfigurasi Tailwind CSS.

---

# 32. Tailwind CSS Strategy

Tailwind CSS digunakan sebagai styling framework utama.

Prinsip:

- Gunakan utility classes.
- Hindari inline style berlebihan.
- Gunakan reusable components.
- Gunakan design tokens.
- Gunakan class composition bila diperlukan.
